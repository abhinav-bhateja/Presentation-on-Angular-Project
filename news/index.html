Student Details

Student Details to Use

  • Roll Number (Required)

Validations #1

Validations #1

Home page of the application should display a list of all the branches. Upon selecting any branch, list of major projects under the selected branch should be displayed. Projects should be highlighted based upon their status. On clicking any project, student details associated with the project should be displayed.

Validations #2

Validations #2

Once a student has been allotted any project, Admin should not be able to assign that student to any other project. Project Name should be unique. A project can have maximum four students and minimum two students. Home page should also contain the admin login facility, from where an admin can add, update and delete any major project.
