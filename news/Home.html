The Navigation Bars

We Used A Side Navigation Bar that was built using Bootstrap and CSS, with a theme of Red/Maroon color inspired by The NorthCap University Logo. It features a collapsible Dropdown with a transition effect that pushes down other content.

The Side Nav itself is also collapsible and is controlled by a FontAwesome Dynamic JS Icon that transitions between the infamous hamburger and a close button which appears on top navbar. The top navbar floats the official logo for our Project Management System. made using Photoshop.
